# BayesNet

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Bayesian Network Classifiers using libtorch from scratch

### Release

```bash
make release
make buildr
```

### Debug & Tests

```bash
make debug
make test
make coverage
```

### Sample app

```bash
make release
make sample
make sample fname=tests/data/glass.arff
```
